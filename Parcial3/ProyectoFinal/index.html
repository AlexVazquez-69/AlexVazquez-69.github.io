Consulta #1. Obtener listado de todos los alumnos de la facultad
SELECT * FROM alumno;
Consulta #2. Obtener listado de cursos (nombre) y todas las asignaturas que tiene
SELECT curso.nombreDescriptivo, asignatura.nombre FROM curso INNER JOIN asignatura ON curso.idCurso=asignatura.curso
Consulta #3. Obtener listado de alumnos (Nombre y apellido) de alumnos que tengan beca
SELECT alumno.apellido1, alumno.nombre FROM alumno WHERE beca LIKE "si"
Consulta #4. Obtener listado de profesores (Nombre y apellido) y numero de telefono
SELECT profesor.nombre apellido1, apellido2 , tlfcontactoprof.telefono FROM profesor INNER JOIN tlfcontactoprof ON profesor.idProfesor = tlfcontactoprof.idProfesor
Consulta #5. Obtener listado de alumnos (todos los campos) que estan inscritos a la asignatura de programacion
SELECT * FROM alumno INNER JOIN asignatura ON asignatura.idAsignatura LIKE "AS009"
